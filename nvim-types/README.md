# nvim-types
Write your [neovim](https://github.com/neovim/neovim) settings in typescript.

## Install 
1. Install this package

```
yarn add -D lua-types
# or
npm install -D lua-types
```

2. Modify your tsconfig.json
```
{
  "compilerOptions": {
    "types": ["nvim-types"]
  }
}
```
